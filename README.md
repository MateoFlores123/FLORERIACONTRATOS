# 🌸 Florería Shalom — Sistema de Contratos

Sistema web para generar contratos de arreglos florales de forma profesional.

## 📁 Estructura del proyecto

```
floreria-shalom/
├── index.html          # Página principal
├── css/
│   └── styles.css      # Estilos (tema terracota/floral)
├── js/
│   └── app.js          # Lógica del formulario y generación del contrato
└── README.md
```

## ✨ Funcionalidades

- Datos del cliente (tratamiento Sr./Sra./Srta., nombre, DNI)
- Fecha y colores del evento
- Múltiples secciones de arreglos (Local, Iglesia, etc.)
- Tabla de items con cantidad, descripción, precio unitario y total
- Transporte y descuento
- Cálculo automático del saldo
- Datos bancarios BCP (CCI y cuenta corriente)
- Vista previa e impresión en formato profesional con logo

## 🚀 Despliegue en Vercel

1. Sube este repositorio a GitHub
2. Ve a [vercel.com](https://vercel.com) e importa el repositorio
3. Vercel detectará automáticamente que es un sitio estático
4. Haz clic en **Deploy** — ¡listo!

No requiere ninguna configuración adicional (sin backend, sin variables de entorno).

## 💻 Uso local

Simplemente abre `index.html` en tu navegador.
